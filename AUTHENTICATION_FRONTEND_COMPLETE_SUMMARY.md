```typescript
authApi.getCurrentUser()  // 获取当前用户

authApi.logout()         // 登出

// 会话管理
sessionApi.getCurrentSession()        // 获取当前会话

// MFA管理
mfaApi.generateMfaSetup()   // 生成MFA设置

mfaApi.verifyMfaCode()      // 验证MFA码
mfaApi.disableMfa()         // 禁用MFA
mfaApi.generateBackupCodes() // 生成备用码

// 审计日志
auditLogApi.queryAuditLogs()      // 查询日志
auditLogApi.statisticsAuditLogs() // 统计日志
auditLogApi.exportAuditLogs()     // 导出日志
```

## 🚀 部署和使用

### 后端部署
```bash
# 1. 数据库初始化
mysql -u root -p data_asset_security < backend/src/main/resources/db/auth_tables.sql

# 2. 启动后端服务
cd backend
mvn clean install
mvn spring-boot:run
```

### 前端部署
```bash
# 1. 安装依赖
cd frontend
npm install

# 2. 启动开发服务器
npm run dev

# 3. 或构建生产版本
npm run build
```

### 访问地址
- **前端应用**: `http://localhost:5173`
- **后端API**: `http://localhost:8080/api`
- **API文档**: `http://localhost:8080/api/doc.html`

5. 输入验证码确认设置
6. 保存备用码

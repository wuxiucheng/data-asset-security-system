# 🎉 数据资产安全及分类分级管理系统

> 企业级数据资产安全管理解决方案，助力企业实现数据资产的全生命周期管理和安全保护

## 📋 项目简介

数据资产安全及分类分级管理系统是一个基于 **Vue 3.4.x** 构建的企业级数据资产管理平台，**前端开发已完成，可以完美体验**！

### 核心特性

- 🎨 **现代化UI界面**：基于Element Plus 2.6.1的精美用户界面
- 🔐 **完善的权限体系**：基于RBAC的三权分立设计，确保数据安全
- 🏗️ **完整的责任体系**：部门和责任人管理，明确数据资产责任归属
- 📊 **细粒度分类分级**：支持字段级别的数据分类分级管理
- 🔄 **灵活的审批流程**：基于Flowable工作流引擎，支持复杂审批场景
- 📈 **强大的统计分析**：多维度数据统计分析和可视化展示
- 📝 **完整的审计日志**：所有关键操作的完整记录和追踪
- ✅ **最新组件规范**：所有Element Plus组件已更新至最新推荐用法

- **端口被占用**: `lsof -i :8080` 或 `lsof -i :5173` 查看端口占用
- **前端无法访问**: 检查Node.js和npm是否正确安装
- **后端无法访问**: 检查Node.js服务是否正常启动

### 📊 数据资产对外输出报告需求清单

根据数据资产分类分级管理的业务需求，以下是对外输出报告的核心功能需求：

#### 1. 数据资产清单报告
**需求描述**: 提供完整的数据资产清单报告，包含所有数据资产的详细信息
**报告内容**:
- 资产基本信息（名称、编码、类型、所属系统）
- 资产责任人信息（部门、责任人、联系方式）
- 分类分级信息（分类、分级、安全级别）
- 资产状态信息（状态、创建时间、最后更新时间）
- 字段级分类分级详情

**输出格式**: Excel、PDF
**生成频率**: 按需生成、定期自动生成
**权限要求**: 数据管理员、系统管理员

#### 2. 数据分类分级统计报告
**需求描述**: 按照分类分级标准统计各类数据的分布情况
**报告内容**:
- 按分类标准统计的资产数量分布
- 按分级标准统计的资产数量分布
- 各部门数据分类分级情况统计
- 分类分级覆盖率统计
- 高级别数据资产清单

**输出格式**: Excel、PDF、HTML
**图表展示**: 饼图、柱状图、树形图
**生成频率**: 每月、每季度、每年
**权限要求**: 所有授权用户

#### 3. 数据安全风险评估报告
**需求描述**: 基于分类分级结果，评估数据安全风险状况
**报告内容**:
- 高级别数据资产风险评估
- 敏感数据分布情况
- 数据访问权限分析
- 数据流转安全状况
- 安全防护建议

**输出格式**: PDF、Word
**生成频率**: 每季度、每半年
**权限要求**: 安全管理员、系统管理员

#### 4. 数据资产责任分布报告
**需求描述**: 统计各部门和责任人的数据资产责任分布
**报告内容**:
- 各部门数据资产数量统计
- 各责任人数据资产数量统计
- 数据资产责任矩阵
- 责任人履职情况分析
- 责任变更历史记录

**输出格式**: Excel、PDF
**生成频率**: 每月、每季度
**权限要求**: 部门管理员、系统管理员

#### 5. 数据资产变更历史报告
**需求描述**: 记录数据资产的变更历史，提供审计追踪
**报告内容**:
- 资产创建、变更、删除历史
- 分类分级变更记录
- 责任人变更记录
- 变更操作审计日志
- 变更影响分析

**输出格式**: Excel、CSV
**生成频率**: 按需查询
**权限要求**: 审计管理员、系统管理员

#### 6. 数据资产使用情况报告
**需求描述**: 统计数据资产的访问和使用情况
**报告内容**:
- 资产访问频次统计
- 资产访问用户分布
- 资产访问时间分析
- 热门资产排行
- 冷门资产识别

**输出格式**: Excel、PDF
**图表展示**: 折线图、热力图
**生成频率**: 每月、每季度
**权限要求**: 数据管理员、系统管理员

#### 7. 合规性检查报告
**需求描述**: 检查数据资产管理是否符合相关法规要求
**报告内容**:
- 数据分类分级合规性检查
- 数据访问权限合规性检查
- 数据安全措施合规性检查
- 审计日志完整性检查
- 合规性风险评估

**输出格式**: PDF、Word
**生成频率**: 每季度、每年
**权限要求**: 合规管理员、系统管理员

#### 8. 数据资产价值评估报告
**需求描述**: 评估数据资产的业务价值和重要性
**报告内容**:
- 数据资产价值评估模型
- 高价值资产识别
- 数据资产价值分布
- 数据资产ROI分析
- 数据资产投资建议

**输出格式**: PDF、Excel
**生成频率**: 每半年、每年
**权限要求**: 高级管理人员、系统管理员

## 📁 技术实现方案

### 前端技术栈
- **报告生成**: Vue 3 + Element Plus
- **图表展示**: ECharts 5.x
- **文档预览**: Vue-PDF
- **文件下载**: Axios + FileSaver.js

### 后端技术栈
- **Excel生成**: EasyExcel 3.3.2
- **PDF生成**: iText 7.x 或 Apache PDFBox
- **图表生成**: JFreeChart 或 ECharts服务器端渲染
- **模板引擎**: Thymeleaf（用于复杂报告模板）

### 数据库设计
**新增表结构**:
```sql
-- 报告模板表
CREATE TABLE report_template (
    template_id BIGINT PRIMARY KEY AUTO_INCREMENT,
    template_name VARCHAR(100) NOT NULL,
    template_type VARCHAR(50) NOT NULL,
    template_content TEXT,
    created_by BIGINT,
    created_time DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_by BIGINT,
    updated_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);

-- 报告生成记录表
CREATE TABLE report_generation_log (
    log_id BIGINT PRIMARY KEY AUTO_INCREMENT,
    report_type VARCHAR(50) NOT NULL,
    report_name VARCHAR(200),
    generation_params TEXT,
    file_path VARCHAR(500),
    file_size BIGINT,
    generated_by BIGINT,
    generation_time DATETIME DEFAULT CURRENT_TIMESTAMP,
    download_count INT DEFAULT 0
);

-- 报告调度配置表
CREATE TABLE report_schedule (
    schedule_id BIGINT PRIMARY KEY AUTO_INCREMENT,
    report_type VARCHAR(50) NOT NULL,
    schedule_name VARCHAR(100),
    cron_expression VARCHAR(100),
    schedule_params TEXT,
    is_active TINYINT DEFAULT 1,
    created_by BIGINT,
    created_time DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

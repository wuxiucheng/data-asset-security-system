# 快速启动指南

## 一键启动

### 启动所有服务（Spring Boot后端 + 前端）
```bash
./start-all.sh
```

### 启动所有服务（Mock后端 + 前端）
```bash
./start-all.sh mock
```

## 常用命令

### 服务管理
```bash
./start-real.sh       # 启动Spring Boot后端
./start-mock.sh       # 启动Mock后端
./start-frontend.sh   # 启动前端
./start-all.sh        # 一键启动（Spring Boot + 前端）

./stop-backend.sh     # 停止后端
./stop-frontend.sh    # 停止前端
./stop-all.sh         # 一键停止

./restart.sh          # 重启Spring Boot后端
./restart.sh mock     # 重启Mock后端

./status.sh           # 查看所有服务状态
```

### 数据库初始化
```bash
# 首次使用需要初始化数据库
mysql -uroot -p1q2w3e4r < backend/create-tables-complete.sql
mysql -uroot -p1q2w3e4r < backend/sync-mock-data.sql
```

### 数据同步
```bash
# 从Mock同步数据到MySQL
node backend/extract-all-mock-data.js > /tmp/all-mock-data.json
python3 backend/generate-full-sync-sql.py
mysql -uroot -p1q2w3e4r < backend/sync-mock-data.sql
```

## 系统要求

- **Java**: 17+（Spring Boot后端）
- **Node.js**: 18+（前端和Mock后端）
- **MySQL**: 8.0+（Spring Boot后端）
- **Maven**: 3.6+（编译Spring Boot）

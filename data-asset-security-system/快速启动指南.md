# 快速启动指南

## 🚀 一键启动

### Linux/Mac 系统
```bash
./start.sh
```

### Windows 系统
```bash
start.bat
```

## 📋 启动命令

### Linux/Mac
```bash
./start.sh start      # 启动所有服务
./start.sh stop       # 停止所有服务
./start.sh restart    # 重启所有服务
./start.sh status     # 查看服务状态
./start.sh logs       # 查看所有日志
./start.sh install    # 安装依赖
./start.sh help       # 显示帮助信息
```

### Windows
```cmd
start.bat start       # 启动所有服务
start.bat stop        # 停止所有服务
start.bat restart     # 重启所有服务
start.bat status      # 查看服务状态
start.bat logs        # 查看所有日志
start.bat install     # 安装依赖
start.bat help        # 显示帮助信息
```

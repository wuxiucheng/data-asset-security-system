# 前端问题修复计划

## 问题分类

### 1. 字段名不匹配问题
**原因**: 前端发送的字段名与后端DTO字段名不一致

**已发现的问题**:
- RoleCreateDTO: 前端发送`description`,后端期望`roleDescription`
- 其他DTO也有类似问题

**解决方案**: 在DTO字段上添加`@JsonAlias`注解

### 2. LocalDateTime序列化问题
**问题**: 会话管理页面报错 - LocalDateTime无法序列化

**解决方案**: 
- 添加Jackson JSR310模块配置
- 或在LocalDateTime字段上添加`@JsonFormat`注解

### 3. API路径问题
**问题**: 部分API返回405错误

**已检查**:
- UserController: ✅ 已有兼容路径
- RoleController: ✅ 已有兼容路径
- 其他Controller: ✅ 已有兼容路径

### 4. 筛选框查询无效
**问题**: 所有页面的筛选框查询无效

**已修复原因**: 前端getList方法只传递page和size参数，未传递筛选条件参数
**解决方案**: 修改前端API的getList方法，使用`{ ...params, page, size }`展开所有参数
